# Dice Roller

## 📌 Description
A simple Python program that simulates rolling a dice and generates a random number between 1 and 6.

## 🚀 Features
- Generates random dice values
- Easy to use
- Interactive output

## 🛠️ Technologies Used
- Python (random module)

## ▶️ How to Run
python main.py
